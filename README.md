# 📊 Python EDA Projects

A collection of **3 Exploratory Data Analysis (EDA)** projects built using Python. Each project dives into a unique real-world dataset to uncover trends, patterns, and insights through data cleaning, analysis, and visualization.

---

## 🗂️ Projects Overview

| # | Project | Dataset | Focus |
|---|---------|---------|-------|
| 1 | 🛍️ Shopping Trends Analysis | `shopping_trends.csv` | Customer purchasing behavior |
| 2 | 🪔 Diwali Sales Analysis | `Diwali Sales Data.csv` | Festive season sales insights |
| 3 | 📚 Book Store Analysis | `Books_Data_Clean.csv` | Book sales, ratings & revenue |

---

## 📁 Project Details

### 1. 🛍️ Shopping Trends Analysis
**File:** `Shopping_trends.ipynb`

Analyzes customer shopping behavior and purchase patterns from a retail dataset.

**Key Analyses:**
- Age distribution of shoppers (Histogram)
- Gender distribution across purchases (Bar Chart)
- Average purchase amount by product category
- Subscription status breakdown (Pie Chart)
- Top colors preferred by customers
- Review ratings analysis

**Libraries Used:** `pandas`, `matplotlib`, `seaborn`, `numpy`

---

### 2. 🪔 Diwali Sales Analysis
**File:** `Diwali_Sales_Analysis.ipynb`

Explores sales data from the Diwali festive season to identify the most valuable customer segments and top-performing regions.

**Key Analyses:**
- Gender-wise buyer count and total spending
- Age group vs. purchase amount (with gender breakdown)
- Top 10 states by total orders and revenue
- Marital status and its influence on purchasing power
- Occupation-wise sales distribution (IT, Healthcare, Aviation, etc.)

**Key Insights:**
- 🔍 Most buyers are **females aged 26–35**
- 📍 **Uttar Pradesh, Maharashtra & Karnataka** lead in orders and revenue
- 💼 Buyers from **IT, Healthcare & Aviation** sectors spend the most

**Libraries Used:** `pandas`, `numpy`, `matplotlib`, `seaborn`

---

### 3. 📚 Book Store Analysis
**File:** `Book_Store.ipynb`

Investigates a book store dataset to understand genre trends, author performance, and what drives book sales and ratings.

**Key Analyses:**
- Distribution of books by publishing year
- Genre-wise book count (Bar Chart)
- Top-rated authors by average rating
- Box plot of rating count per genre
- Sale price vs. units sold (Scatter Plot)
- Language distribution of books (Pie Chart)
- Top authors by gross sales revenue
- Trend of units sold over publishing years (Line Chart)
- Author rating vs. units sold comparison

**Libraries Used:** `pandas`, `matplotlib`, `seaborn`

## 🚀 Getting Started

### Prerequisites
Make sure you have Python and Jupyter Notebook installed.

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Run the Notebooks

```bash
# Clone the repository
git clone https://github.com/your-username/your-repo-name.git

# Navigate to the project folder
cd your-repo-name

# Launch Jupyter Notebook
jupyter notebook
```

Then open any `.ipynb` file and run the cells sequentially.
